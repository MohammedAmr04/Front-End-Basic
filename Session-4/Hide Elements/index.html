Hiding Elements in CSS

1️⃣ opacity: 0

I am invisible but still here!

⚠️ The element is still in the layout, and you can interact with it.

2️⃣ visibility: hidden

I am hidden but still taking up space!

⚠️ The element is hidden, but its space remains.

3️⃣ display: none

I am completely removed!

✅ The element is gone, and it does not take up any space.
